## OpenCV installation

OpenCV installation guide for MacOS

1. mkdir open_cv <br />
2. git clone https://github.com/opencv/opencv.git <br />
3. mkdir build <br />
4. cd ./build <br />
5. cmake ../opencv/ . <br />
6. sudo make -j 4 <br />
7. sudo make install <br />

Cmake installation
1. brew install cmake
